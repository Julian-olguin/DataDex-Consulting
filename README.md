# 📊 DataDex Consulting
### Proyecto: Análisis y Optimización de Datos - Caso "Enciclopedia Pokémon"

## 📝 Descripción del Proyecto
Este repositorio contiene el desarrollo integral de una solución de bases de datos y análisis de información para el caso de estudio "Enciclopedia Pokémon". El objetivo principal del proyecto es demostrar cómo organizar, limpiar y estructurar grandes volúmenes de datos en bruto para realizar un análisis eficiente y extraer información estratégica.

El proyecto abarca desde el diseño lógico de la información hasta la generación de visualizaciones a través de consultas en bases de datos relacionales.

## 🎯 Objetivos y Alcance
1. **Modelado y Arquitectura:** Diseño de un esquema conceptual detallado y Diagrama Entidad-Relación (DER).
2. **Procesamiento de Datos:** Aplicación de técnicas de limpieza, normalización y optimización sobre un conjunto de datos inicial (CSV).
3. **Gestión de Base de Datos:** Generación de archivos normalizados listos para la creación de la base de datos y el llenado eficiente de sus tablas.
4. **Análisis y Visualización:** Desarrollo de scripts para generar gráficas analíticas sobre el rendimiento, captura y combate de los Pokémon.

## 📂 Estructura del Repositorio
Para mantener un entorno de trabajo organizado, el proyecto se divide en:
- `/docs`: Documentación técnica y diagramas de la base de datos.
- `/scripts`: Código fuente utilizado para la limpieza y normalización de los datos originales.
